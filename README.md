# Explorando Padrões de Projeto em Java

## Descrição

Projeto desenvolvido como desafio do Bootcamp da DIO para praticar a implementação dos padrões de projeto utilizando Java.

## Padrões implementados

- Singleton
- Strategy
- Facade

## Melhoria realizada

Além da implementação original, foi adicionado um novo comportamento ao padrão Strategy:

- ComportamentoCauteloso

Essa melhoria demonstra como o padrão Strategy permite adicionar novos comportamentos sem alterar a estrutura principal da aplicação.

## Tecnologias

- Java 21
- IntelliJ IDEA
- Git
- GitHub

## Como executar

1. Clone o repositório.
2. Abra o projeto no IntelliJ IDEA.
3. Execute a classe `Test.java`.
